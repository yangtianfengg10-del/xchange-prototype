How it works
All guides are shown here (as requested). This is the prototype behaviour:
1) Login
Prototype uses 3 pre-made accounts. After login, you’ll see a badge showing your user type.
2) Coins
New users start with 50 coins. 1 coin = $1. Upgrade → choose coin pack → Transaction → Purchase (adds coins).
3) Choose Skills
Skills page has two categories: Learn and Exchange. Filter by Video or Live sessions.
4) Buy Learn courses
Coins are deducted. If it’s a Video course, you’re automatically added to its private group. If it’s Live, a scheduling request is created.
5) Learning groups
Go to Groups page to see your private groups and watch videos (demo player).
6) Teacher management
Teacher / Teacher&Student can manage groups and add videos manually in the Teacher Dashboard.
7) Exchange courses
Only Teacher&Student users can access Exchange. Both users must select each other to match. Cheaper-course owner pays the difference in coins.
8) Chat
Chat is available only when signed in. Messages are stored locally in your browser.
9) Upgrade
Free plan limits teaching; Creator plan unlocks more groups/courses.
Plans
Free plan: teachers can create up to 2 courses and 3 groups (limited).
Creator ($8/month): teachers can create up to 10 courses and 10 groups.
Campus custom: for universities/schools (business discussion).
