Learn • Teach • Exchange

Xchange — a coin-based skill marketplace where students and teachers match.

1 coin = $1. New users start with 50 coins. Buy affordable courses, join private video groups, or schedule live sessions.

Overview
Xchange supports three user roles: Student, Teacher, and Teacher & Student. Roles are shown as a badge after login.

Features
Learn (Video / Live)
Buy video classes to join private groups. Live classes create a scheduling request.
Exchange (Match-based)
Only Teacher&Student users can exchange. Both sides must select each other.
Upgrade + Transaction
Buy coins → Transaction page → coins added automatically for demo.
What you can demo
Login, browse skills, buy coins via transaction, upgrade plan, buy courses, check groups & videos, manage groups as teacher, try exchange matching, and chat. All data persists in your browser.
